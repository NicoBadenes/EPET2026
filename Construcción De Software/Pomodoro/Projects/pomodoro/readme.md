# Comodoro App

## Descripción

**Comodoro App** es una aplicación móvil simple hecha con **React Native y Expo** que permite usar un temporizador estilo **Pomodoro** para mejorar la productividad.  

El usuario puede iniciar, pausar y reiniciar ciclos de trabajo y descanso.

---

## Características

- Temporizador **Pomodoro** (25 min de trabajo / 5 min de descanso o 10 min de descanso)  
- Visualización del tiempo restante  
- Botones para **Iniciar**, **Pausar** y **Reiniciar**  
- Diseño simple y minimalista  
- Compatible con **iOS** y **Android** mediante Expo Go

---

## Tecnologías

- [React Native](https://reactnative.dev/)  
- [Expo](https://expo.dev/)  
- JavaScript (ES6+)  
- `StyleSheet` para estilos nativos
